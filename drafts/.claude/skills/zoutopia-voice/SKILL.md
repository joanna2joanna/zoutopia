# 走托邦 · 写作DNA

> 从 article-1 到 article-5 的定稿中反向蒸馏。
> 这是 Joanna 和 Claude 共同维护的写作风格。

## 身份

Joanna：走托邦作者，INFJ，文旅创意从业者，200潜潜水员，广东人。
写作是内心独白——写给自己的文字恰好被人读到。

## 五层DNA

### 怎么说话（表达DNA）

**句式指纹：**
- 每行≤22字，以逗号为自然断点
- 短行短段，手机竖屏友好
- 句号少，逗号多——话没说完不急着断
- 段落收尾不设金句，轻轻放下

**词汇指纹：**
- 具体名词代泛指：耀华力路 > 唐人街，知子罗 > 废弃县城
- 动词加一层意思："到了"→"来到"，"看到"→"初见"
- "的"和"之"轮换，"和"与"与"交替
- 不写"忽然""突然""真正""早已""顷刻""闪现"——一篇不超过两个

**节奏指纹：**
- 不写排比三连击
- 不写"不是……而是……"式AI对仗
- "越……越……"一篇只用一次
- 直接说人话，不绕比喻——七个字>"一杆秤"绕三行

### 怎么想（心智模型）

**1. 观察者视角，不下判断。**
不是评论家，是站在水里的另一个人。说"噢，你也在"，不说"你应该"。
案例：article-5 把上山下乡/双创/OPC摆在一起，不问好坏，只问"为什么砸中了同一批人"。

**2. 用画面说话，不替读者感受。**
给画面，不接旁白。读者自己会怕、会感动。
案例：article-3 沉船夜潜——光束太小、沉船太大、盲人摸象——没有"那一刻心脏狂跳"。

**3. 数据为骨，感受为肉。**
数字先立住，然后退一步说"我看不懂"或"暂且不想"。
案例：article-5 三组数据→"原因是什么？不敢多想"。

**4. 以"我"为锚，不站到读者那边去。**
全文统一主观视角。"我怕"→从头到尾都是"我"。
案例：article-3 全文第一人称恐惧叙述，读到结尾也不换视角。

**5. callback不是解释，是确认。**
点到为止，一两句够了。读者不需要你替他回忆。
案例：article-1 "文化孤岛"种下→曼谷回收→知子罗回收→月球回收。

### 怎么判断（决策启发式）

- 如果一段话读起来像金句，删掉。
- 如果一组对比能用两个数据说完，不加第三个。
- 如果比喻来自脑子里的"这个好像挺漂亮"，删掉——只留身体记忆里的画面。
- 如果写完觉得"这个结尾好有力"，大概率是AI味——换一个轻的。
- 如果数据来自模糊来源（"研究表明""宏观数据说"），要么给来源，要么删。

### 不做什么（反模式）

- **不写金句。** 任何听起来像可以被单独截图引用的话，重写。
- **不排比三连。** 两个够了，三个是AI。
- **不替历史/制度/读者做判断。** 只说"我看到"，不说"这说明"。
- **不表演共情。** 不写"你也是这样吧""我们都……"。读者自己会对号入座。
- **不在文末总结。** 结尾是呼吸，不是收网。"我还没有"不是结论。
- **不用模糊引述。** "宏观数据说""研究表明"直接删。
- **不用比喻堆砌。** 一个比喻就够。直接说人话最有力量。

### 每篇DNA分解

- **article-1《算法，史诗，与童年》**：粤语观察、文化孤岛、知子罗、691基地、图卢姆——全是你的行李。拆行和"的/之"轮换是AI，但骨架血肉都是你的。callback网络（文化孤岛→曼谷→知子罗→月球撞击坑）你埋种子，AI浇水。
- **article-2《如果宇宙是造物主的过家家》**：AI味最重的一篇。分辨率三段展开、外星人三种可能、素材密度高切换快——结构太整齐，呼吸感被压住了。开篇"小红书冲浪"的口吻是你的。**不是失败——它帮你认清了什么不是你。**
- **article-3《那些源于恐惧的热爱》**：你最纯粹的声音，是衡量其他篇AI味的基线。潜水的身体记忆、凌云群峰、高美湿地风车、沉船夜潜、浅水浸死人——AI几乎为零。**这篇就是基准。**
- **article-4《AI最好的善意是释放思考》**："先交个底"的口吻、旅行者号类比（飞了四十八年，还没走到一光天）、"草率结论"收尾是你的。四段平行罗列（case-study/project-study/observation-note/zoutopia）AI痕迹偏重。
- **article-5《高考题外话》**：我们合作最平衡的一篇。上山下乡→双创→OPC的跳跃、数据追问、"暂且不想"的位置是你的。三组数据→三个念头结构是AI。洪流callback和"我还没有"的落点是混合的。

### 知道局限（诚实边界）

- 不能替代 Joanna 的真实经历和身体记忆。AI 永远写不出"沉船夜潜"和"跳楼机测评"。
- 过于复杂的数据论证会杀死鸡皮疙瘩。数据是骨架，感受是呼吸——骨架整齐就行，不用解剖。
- 有些文章（如 article-3）几乎不依赖 AI，纯粹个人表达。这类文章不要用这个 skill 去动。
- 微信平台发布有独立的敏感词过滤需求，和写作美学无关。发之前跑 grep。
